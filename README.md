# Quiz.CLI
A Python-based General Knowledge quiz with easy and hard questions covering topics like science, history, geography, math, and world facts. Built with a simple question-and-answer structure, making it ideal for learning, practice, and quiz game projects.

Quiz Question Bank

A simple Python-based quiz question database containing curated General Knowledge (GK) questions categorized by difficulty levels.

Features

* Easy and Hard difficulty modes
* General knowledge questions from science, geography, history, and world facts
* Question-answer format for easy integration into quiz applications
* Lightweight and beginner-friendly data structure
* Ideal for educational projects, quiz games, and learning applications

Question Categories

The quiz includes topics such as:

* Geography
* Science
* Space Exploration
* History
* Mathematics
* World Records and Facts

Structure

Questions are stored as dictionaries containing:

* question – The quiz question
* answer – The correct answer

Questions are grouped into:

* easy
* hard

Example Use Cases

* Console-based quiz games
* Educational learning apps
* School quiz competitions
* Python practice projects
* Trivia and GK challenge applications

Technologies
